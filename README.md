# NAME

Alien::Z3 - Perl distribution for Z3

# VERSION

version 0.001

# INSTALL

    cpanm Alien::Z3

# DESCRIPTION

This Alien module wraps the Z3 C library. This release supports version 4.8.4 of the Z3 library
